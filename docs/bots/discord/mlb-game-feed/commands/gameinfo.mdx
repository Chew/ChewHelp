# `/gameinfo` Command

The `/gameinfo` command shows you information about any game for the day. It will be able to show you boxscore information and more.

All responses are ephemeral, meaning only you can see them.

## Usage

Start by typing `/gameinfo` and selecting a game to see the information for that game.

![example](https://cdn.chew.pro/imgs/lInstpf.png)

From there, use the dropdown to select information for that team.

![example](https://cdn.chew.pro/imgs/YAnKN7b.png)

### Viewing the Lineup

You can view the lineup for the game by selecting the "Lineup" option from the menu.

:::note

This is the same response as clicking the [`/plangame`](./plangame) lineup buttons.

:::

![example lineup](https://cdn.chew.pro/imgs/e7l8II9.png)

### Viewing the Box Score

You can view the box score for the game by selecting the "Box Score" option from the menu.
