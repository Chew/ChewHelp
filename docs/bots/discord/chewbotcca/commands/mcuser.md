# `/mcuser` Command

Looks up a Minecraft user and returns their profile

## Arguments

| Name | Type   | Kind     | Description                     |
|------|--------|----------|---------------------------------|
| user | String | Required | The user name or UUID to lookup |

## Permissions

Bots need the following permissions: Embed Links.
