# `/rubygem` Command

Searches for and returns some basic info about a specified ruby gem

## Arguments

| Name | Type   | Kind     | Description       |
|------|--------|----------|-------------------|
| gem  | String | Required | The gem to lookup |

## Permissions

Bots need the following permissions: Embed Links.
