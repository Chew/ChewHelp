# `/info` Command

Returns some detailed information about a specified command

## Arguments

| Name    | Type   | Kind     | Description                  |
|---------|--------|----------|------------------------------|
| command | String | Required | The command to find info for |

## Permissions

Bots need the following permissions: Embed Links.
