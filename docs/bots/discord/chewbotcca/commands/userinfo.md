# `/userinfo` Command

Returns some useful info about you or another user

## Arguments

| Name | Type | Kind     | Description        |
|------|------|----------|--------------------|
| user | User | Optional | The user to lookup |

## Permissions

The bot does not need any special permissions to run this command.
