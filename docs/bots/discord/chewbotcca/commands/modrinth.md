# `/modrinth` Command

Searches modrinth.com for a specified project

## Arguments

| Name  | Type   | Kind     | Description               |
|-------|--------|----------|---------------------------|
| query | String | Required | The project to search for |

## Permissions

The bot does not need any special permissions to run this command.
