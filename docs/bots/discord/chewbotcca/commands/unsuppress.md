# `/unsuppress` Command

Unsuppresses an embed given a message URL

## Arguments

| Name         | Type   | Kind     | Description               |
|--------------|--------|----------|---------------------------|
| message_link | String | Required | The message to unsuppress |

## Permissions

Bots need the following permissions: Embed Links.
