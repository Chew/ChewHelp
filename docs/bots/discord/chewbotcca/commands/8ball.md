# `/8ball` Command

Ask the magic 8ball a question!

## Arguments

| Name     | Type   | Kind     | Description                        |
|----------|--------|----------|------------------------------------|
| question | String | Required | The question to ask the eight ball |

## Permissions

The bot does not need any special permissions to run this command.
