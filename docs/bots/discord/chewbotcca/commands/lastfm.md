# `/lastfm` Command

Returns playing status for a specified user

## Arguments

| Name     | Type   | Kind     | Description                                 |
|----------|--------|----------|---------------------------------------------|
| username | String | Optional | The user to look up (default: yours if set) |

## Permissions

The bot does not need any special permissions to run this command.
