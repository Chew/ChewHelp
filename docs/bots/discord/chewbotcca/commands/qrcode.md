# `/qrcode` Command

Converts arguments into a QR Code

## Arguments

| Name | Type   | Kind     | Description                        |
|------|--------|----------|------------------------------------|
| data | String | Required | The data to convert into a QR Code |

## Permissions

Bots need the following permissions: Embed Links.
