# `/mcissue` Command

Searches Mojira or Spigot MC Jira for a specified issue. Requires PROJ-NUM or link

## Arguments

| Name  | Type   | Kind     | Description         |
|-------|--------|----------|---------------------|
| issue | String | Required | The issue to lookup |

## Permissions

Bots need the following permissions: Embed Links.
