# `/youtube` Command

Queries YouTube for a video

## Arguments

| Name  | Type   | Kind     | Description                          |
|-------|--------|----------|--------------------------------------|
| query | String | Required | The query to search for videos with. |

## Permissions

Bots need the following permissions: Embed Links.
