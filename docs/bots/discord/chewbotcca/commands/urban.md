# `/urban` Command

Checks a word on Urban Dictionary. Only works in NSFW channels

## Arguments

| Name | Type   | Kind     | Description        |
|------|--------|----------|--------------------|
| word | String | Required | The word to define |

## Permissions

The bot does not need any special permissions to run this command.
