# `/mcwiki` Command

Search the Minecraft Wiki

## Arguments

| Name  | Type   | Kind     | Description                     |
|-------|--------|----------|---------------------------------|
| query | String | Required | The query to lookup on the wiki |

## Permissions

Bots need the following permissions: Embed Links.
