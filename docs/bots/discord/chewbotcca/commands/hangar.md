# `/hangar` Command

Search for a plugin on [Hangar](https://hangar.papermc.io)

## Arguments

| Name   | Type   | Kind     | Description              |
|--------|--------|----------|--------------------------|
| plugin | String | Required | The plugin to search for |

## Permissions

Bots need the following permissions: Embed Links.
