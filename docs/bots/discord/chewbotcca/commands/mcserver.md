# `/mcserver` Command

Find some information about a specified server

## Arguments

| Name | Type   | Kind     | Description                  |
|------|--------|----------|------------------------------|
| ip   | String | Required | The server IP, port optional |

## Permissions

Bots need the following permissions: Embed Links.
