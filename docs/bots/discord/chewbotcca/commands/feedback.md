# `/feedback` Command

Leave some feedback about the bot

## Arguments

| Name     | Type   | Kind     | Description                    |
|----------|--------|----------|--------------------------------|
| feedback | String | Required | The feedback you want to leave |

## Permissions

The bot does not need any special permissions to run this command.
